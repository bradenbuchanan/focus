## Tech Stack Overview

### Frontend

- **Next.js 14+** (App Router) - For page routing and server components
- **TypeScript** - For type safety and better developer experience
- normal styling, no tailwind
- **React Query or SWR** - For efficient data fetching and caching
- **Framer Motion** - For animations on timers, progress bars, and transitions
- **React Hook Form** - For form handling
- **Chart.js or D3.js** - For analytics visualizations and heatmaps

### Backend

- **Next.js API Routes** - For serverless API endpoints
- **Prisma** - Type-safe ORM for database interactions
- **NextAuth.js** - For authentication system
- **Stripe** - For payment processing (one-time $9.99 payment)
- **Pusher or Socket.io** - For real-time features (group challenges, leaderboards)

### Database

- **PostgreSQL** - Primary database (via Supabase, Neon, or Railway)
- **Redis** - For leaderboards and real-time features (via Upstash)

### Deployment

- **Vercel** - For hosting the Next.js application
- **Supabase or Neon** - For managed PostgreSQL
